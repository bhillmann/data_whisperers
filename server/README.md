> The best module ever.


## Install

```sh
$ npm install --save data-whisperers
```


## Usage

```js
var data-whisperers = require('data-whisperers');

data-whisperers('Rainbow');
```

```sh
$ npm install --global data-whisperers
$ data-whisperers --help
```

```sh
# creates a browser.js
$ npm run browser
```
